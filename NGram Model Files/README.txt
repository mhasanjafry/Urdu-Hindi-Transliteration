This folder constist of two files:
1. ngram_hindi_4.txt: This file consists of character based probabilities for n=1,2,3,4 grams for Hindi. It basically gives the probability of the particular character gram appearing in the training corpus (W2C Hindi).
2. ngram_urdu_4.txt: This file consists of character based probabilities for n=1,2,3,4 grams for Urdu. It basically gives the probability of the particular character gram appearing in the training corpus (W2C Urdu).

We have many such model files but did not upload as the size of the files was huge.
